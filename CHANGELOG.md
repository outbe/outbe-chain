# Changelog

All notable changes to this project are documented here. This file is managed
by [cocogitto](https://docs.cocogitto.io/) (`cog`): each `cog bump --auto`
derives the next SemVer from the Conventional Commits since the last `v*` tag
and prepends the corresponding section.

The baseline tag is `v0.1.0`; versioning and these entries cover commits made
after it.
